Amazon Web Services for Hydrosat Data Analytics Platform (DAP)¶

This website hosts documents, slides, material, Hands-on videos and information related to the tutorial “AWS for Hydrosat DAP” given during the Hydrosat Training from RDI Consultancy.

Covered Topics¶

The objective of this tutorial is to offer a set of practical sessions and tutorials around AWS, for helping Hydrosat deploying their DAP on a cloud environment in order to have a stable and high perfromance work environment. The tutorial is covering the following areas in AWS:

  1. Start working on AWS by creating an account
  2. AWS Computing
  3. Run Python Scripts on EC2
  4. Web servers on EC2 instances
  5. AWS Lightsail
  6. Other Computing Services: Serverless and Containers
  7. Storage volumes in AWS
  8. Networking in AWS
  9. Relational databases in AWS
  10. Load balancing and Scalling in AWS
  11. AWS Pricing and Support
  12. AWS Security and Access Management
